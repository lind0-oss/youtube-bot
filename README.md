# youtube-bot
 bot de views no youtube
# instalação
 Baixe o projeto pelo github, depois instale as dependencias executando o arquivo ``dependencias.cmd`` ou ``dependencias.sh`` logo depois abra o seu terminal e escreva:
 ```
 cd youtube-bot
 cd bot
 python viewers.py
 ```
 pronto.
